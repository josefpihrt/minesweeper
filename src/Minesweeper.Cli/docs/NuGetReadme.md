# dotnet-minesweeper

Play Minesweeper on the command line.

## Installation

```bash
dotnet tool install dotnet-minesweeper -g
```

## Keybindings

| Key | Action |
| --- | --- |
| Enter | Open cell |
| Ctrl+Enter | Flag cell |
| Ctrl+Arrow | Jump to next unknown cell |
| Shift+Arrow | Expand selection |
| P | Pause game |
| Ctrl+C | Cancel game |

## Presets

| --- | --- | --- | --- |
